Read Me

The purpose of this analytics is to discover the relationship between economic index GNI and per visa sponsorships

The majority of analytics is done in hive. Some data cleaning is performed before importing to Hive using map reduce.

To run the analytics, simply take the cleaned dataset in the shared project folder, import them into hive tables, and run the queries as shown in the jury script and screenshot.

The output of the queries are stored in the table directory of the project folder.
